The Core Finding

For twin prime candidates (6k-1, 6k+1), we classify each pair by primality state. The composite member of a PC pair has ~2.93% more distinct prime factors than composites in CC pairs.

Theoretical Derivation

A first-principles derivation in three layers: exact identity, heuristic sum, empirical calibration.

Layer 1: The Exact Local Identity

For each prime p ≥ 5, exactly one residue class k (mod p) has p | (6k-1), and exactly one has p | (6k+1). These are disjoint since p | (b-a) = 2 requires p = 2. So the events "p|a" and "p|b" are mutually exclusive.

P(p|b | p ∤ a) = P(p|b) / P(p ∤ a) = (1/p) / (1-1/p) = 1/(p-1)

This step is exact. Everything else is heuristic.

Layer 2: The Independent-Prime Sum

Per-prime increment: δp = 1/(p-1) - 1/p = 1/[p(p-1)]

∑_{p≥5} 1/[p(p-1)] = 0.1065

This sum converges, explaining why the bias stabilizes.

Layer 3: Empirical Calibration

The naive prediction overshoots. Define calibration factor:

c = Δωemp / 0.1065 = 0.0828 / 0.1065 = 0.78

The ~22% overshoot comes from correlations + CC conditioning.

Paper-Ready Summary

For each prime p ≥ 5, the congruences 6k ≡ ±1 (mod p) have unique solutions, so among k (mod p) exactly one class satisfies p | (6k-1) and exactly one satisfies p | (6k+1). These are disjoint since p | 2 would be required for overlap. Hence conditioning on p ∤ (6k-1) boosts the probability that p | (6k+1) from 1/p to 1/(p-1). Summing the per-prime increment 1/[p(p-1)] over p ≥ 5 yields a convergent absolute shift of 0.1065. Empirically, the observed PC–CC shift at K=109 is 0.083, about 22% smaller than the naive prediction, consistent with residual arithmetic correlations and the fact that CC is itself a conditioned population.
